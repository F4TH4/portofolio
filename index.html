About Me

Mahasiswa AMIK yang memiliki ketertarikan pada pengembangan web, khususnya di bidang frontend. Senang mempelajari cara membangun tampilan website yang rapi dan responsif.
Terbiasa menggunakan HTML, CSS, dan JavaScript dalam pembuatan website statis, serta memahami struktur dasar dan praktik penulisan kode yang bersih.
Memiliki perhatian pada detail desain, termasuk tata letak, warna, dan tipografi, agar tampilan website nyaman digunakan di berbagai perangkat.
Tertarik mendalami UI/UX serta animasi web sederhana untuk meningkatkan pengalaman pengguna tanpa mengorbankan performa.
Mampu bekerja secara mandiri maupun dalam tim kecil, serta terbuka terhadap masukan dan pembelajaran baru di dunia teknologi.
Berkomitmen untuk terus mengembangkan kemampuan dan mengikuti perkembangan terbaru dalam bidang web development.
Memiliki kemampuan problem solving dan analisis yang baik, sehingga dapat menyelesaikan permasalahan pengembangan website dengan cepat dan efektif.
Selalu berupaya mengikuti perkembangan teknologi web terbaru, termasuk framework dan tools modern, agar hasil pengembangan lebih efisien dan sesuai standar industri.
